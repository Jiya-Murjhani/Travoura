# Travel AI App

## Overview
The Travel AI App is a web application designed to assist users in planning their trips using an intuitive interface powered by AI. The app allows users to input their travel preferences, budget, and special requests, and generates a personalized itinerary.

## Features
- **Responsive Design**: The app is fully responsive, providing a seamless experience on both mobile and desktop devices.
- **Trip Planning Interface**: Users can easily navigate through a step-based flow to plan their trips.
- **Dynamic Inputs**: The app includes editable fields for destination, date selection, traveler count, and budget.
- **Accommodation and Transport Preferences**: Users can select their accommodation and transport preferences through interactive cards and dropdowns.
- **Trip Vibe Selection**: Users can choose their desired trip vibe from a set of selectable cards.
- **Live Preview**: A sidebar that dynamically updates to show estimated costs and suggested trip duration based on user input.
- **AI Preferences**: Options to toggle preferences for hidden gems, avoiding crowds, and budget optimization.
- **Smooth Animations**: The app features smooth transitions and micro-animations for a modern user experience.

## Installation
To get started with the Travel AI App, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/travel-ai-app.git
   cd travel-ai-app
   ```

2. **Install Dependencies**:
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the Application**:
   Start the development server:
   ```bash
   npm start
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:3000` to view the app in your browser.

## Usage
- Click the "Start Planning" button on the landing page to begin the trip planning process.
- Fill in the required fields and select your preferences.
- Review the live preview sidebar for estimated costs and suggested duration.
- Submit your trip plan to generate a personalized itinerary.

## Technologies Used
- **React**: For building the user interface.
- **Tailwind CSS**: For styling and responsive design.
- **TypeScript**: For type safety and better development experience.
